# ETF Lens

可直接使用的台灣 ETF 分析儀表板，支援：`0050`、`0052`、`00878`、`00991A`、`00988A`、`00981A`、`00935`、`009816`、`00881`、`00891`、`00941`、`00403A`。

## 使用方式

直接雙擊 `index.html`，或用瀏覽器開啟即可，無需安裝任何套件。

## 功能

- ETF 代號搜尋與輸入提示
- 前十大持股與權重
- 產業配置圓環圖
- 1、3、5 年年化報酬（上市期間不足時顯示資料不足）
- Sharpe Ratio 與最大回撤
- 1、3、5 年績效走勢互動圖
- 響應式手機版面

> 目前內建資料供產品展示與介面測試使用，不構成投資建議。正式上線時應串接投信或證交所的授權資料來源。

## GitHub Pages

推送至 `main` 分支後，GitHub Actions 會自動發布網站。儲存庫的 **Settings → Pages → Source** 請設為 **GitHub Actions**。
